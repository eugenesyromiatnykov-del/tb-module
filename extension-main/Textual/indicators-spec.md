# СПЕЦИФИКАЦИЯ ИНДИКАТОРОВ ДЛЯ РАЗРАБОТЧИКА

## Структура индикатора

Каждый индикатор должен содержать:

```javascript
{
  id: string,                          // Уникальный идентификатор
  name: string,                        // Название индикатора
  category: string,                    // Категория
  type: string,                        // Тип (ОБСТЕЖЕННЯ, НАПРАВЛЕННЯ, ДІАГНОСТИЧНИЙ_ЗВІТ)
  gender: 'M' | 'F' | 'BOTH',         // Пол пациента
  applies: (patient) => boolean,       // Функция проверки применимости
  frequency: (patient) => number,      // Функция возврата частоты (месяцы)
  requiredActions: Array,              // Обязательные действия
  riskFactors: Array,                  // Коды факторов риска
  exclusionCodes: Array,               // Коды исключения
  description: string                  // Описание
}
```

## ИНДИКАТОР 1: ПОВНЕ ОЦІНЮВАННЯ СЕРЦЕ-СУДИННОГО РИЗИКУ

**ID:** `cvd-risk-combined`

### Применяемость
- **Пол:** оба
- **Возраст:** М ≥ 40 лет, Ж ≥ 50 лет
- **Исключение:** не применяется если есть K86 или K87

### Функция applies()
```javascript
applies: (patient) => {
  if (!patient.gender || !patient.age) return false;
  
  const diagnosisCodes = extractDiagnosisCodes(patient.diagnoses);
  
  // ИСКЛЮЧЕНИЕ: гіпертонія
  if (diagnosisCodes.includes('K86') || diagnosisCodes.includes('K87')) {
    return false;
  }
  
  // УСЛОВИЕ: возраст по полу
  if (patient.gender === 'M' && patient.age >= 40) return true;
  if (patient.gender === 'F' && patient.age >= 50) return true;
  
  return false;
}
```

### Функция frequency()
```javascript
frequency: (patient) => {
  // БЕЗ факторов риска → раз в 24 месяца (2 года)
  // С факторами риска → раз в 12 месяцев (1 год)
  
  if (!patient.diagnoses) return 24;
  
  const diagnosisCodes = extractDiagnosisCodes(patient.diagnoses);
  const hasRiskFactor = checkRiskFactors(diagnosisCodes, RISK_FACTORS_CVD);
  
  return hasRiskFactor ? 12 : 24;
}
```

## ІНДИКАТОР 3: СКРИНІНГ НА ВИЯВЛЕННЯ ЦУКРОВОГО ДІАБЕТУ

**ID:** `diabetes-screening`

### Применяемость
- **Пол:** оба
- **Возраст:** без ограничений, если есть факторы риска; ≥ 45 лет без факторов
- **Исключение:** не применяется если есть T89 или T90

### Функция applies()
```javascript
applies: (patient) => {
  if (!patient.diagnoses) return false;
  
  const diagnosisCodes = extractDiagnosisCodes(patient.diagnoses);
  
  // ИСКЛЮЧЕНИЕ: уже есть диабет
  if (diagnosisCodes.includes('T89') || diagnosisCodes.includes('T90')) {
    return false;
  }
  
  // УСЛОВИЕ 1: есть факторы риска (независимо от возраста)
  const hasRiskFactor = checkRiskFactors(diagnosisCodes, RISK_FACTORS_DIABETES);
  if (hasRiskFactor) return true;
  
  // УСЛОВИЕ 2: возраст ≥ 45 лет (без факторов)
  if (patient.age && patient.age >= 45) return true;
  
  return false;
}
```

### Функция frequency()
```javascript
frequency: (patient) => {
  // Раз в год (12 месяцев)
  return 12;
}
```

### Обязательные обстежень

| T34025 | Вимірювання глюкози крові натще |
| 14743-9 | Глюкоза (глюкометр) |

## ІНДИКАТОР 8: СКРИНІНГ НА ВИЯВЛЕННЯ РАКУ МОЛОЧНОЇ ЗАЛОЗИ (НАПРАВЛЕННЯ)

**ID:** `breast-cancer-screening-referral`

### Применяемость
- **Пол:** только Ж
- **Возраст:** 
  - 40 ≤ x ≤ 69 лет (при наличии факторов риска)
  - 50 ≤ x ≤ 69 лет (без факторов)
- **Исключение:** не применяется если есть X76

### Функция applies()
```javascript
applies: (patient) => {
  if (patient.gender !== 'F') return false;
  if (!patient.age) return false;
  if (!patient.diagnoses) return false;
  
  const diagnosisCodes = extractDiagnosisCodes(patient.diagnoses);
  
  // ИСКЛЮЧЕНИЕ
  if (diagnosisCodes.includes('X76')) return false;
  
  const hasRiskFactor = checkRiskFactors(diagnosisCodes, RISK_FACTORS_BREAST);
  
  // УСЛОВИЕ 1: с факторами + 40-69 лет
  if (hasRiskFactor && patient.age >= 40 && patient.age <= 69) return true;
  
  // УСЛОВИЕ 2: без факторов + 50-69 лет
  if (!hasRiskFactor && patient.age >= 50 && patient.age <= 69) return true;
  
  return false;
}
```

### Функция frequency()
```javascript
frequency: (patient) => {
  if (!patient.diagnoses) return 24;
  
  const diagnosisCodes = extractDiagnosisCodes(patient.diagnoses);
  const hasRiskFactor = checkRiskFactors(diagnosisCodes, RISK_FACTORS_BREAST);
  
  // С факторами: раз в год (12 месяцев)
  // Без факторов: раз в 2 года (24 месяца)
  return hasRiskFactor ? 12 : 24;
}
```

- **Условие:** ТОЛЬКО если есть хотя бы один из:

  - K74 или K76 (серцево-судинні захворювання)
  - T89 или T90 (цукровий діабет)

### Функция applies()
```javascript
applies: (patient) => {

  return hypertension || cardiovascular || diabetes;
}
```

## ВСПОМОГАТЕЛЬНЫЕ ФУНКЦИИ

### extractDiagnosisCodes(diagnoses)
Преобразует массив диагнозов в массив кодов:

```javascript
function extractDiagnosisCodes(diagnoses) {
  if (!diagnoses) return [];
  
  return diagnoses
    .map(d => {
      if (typeof d === 'string') return d;
      return d && d.code ? d.code : null;
    })
    .filter(d => d !== null);
}
```

### checkRiskFactors(diagnosisCodes, riskFactors)
Проверяет наличие факторов риска:

```javascript
function checkRiskFactors(diagnosisCodes, riskFactors) {
  return diagnosisCodes.some(code => riskFactors.includes(code));
}
```
